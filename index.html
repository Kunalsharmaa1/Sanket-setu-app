Women Safety Alert Application

Community-Based Emergency Response System

Problem Statement

Women often face unsafe situations where immediate help is required. During emergencies, it is difficult to manually call police or share location. A fast, automated alert system is needed.

Proposed Solution

This project proposes a mobile-based application with an SOS button. When pressed, it sends the user's real-time location to nearby registered users and police authorities for immediate assistance.

System Architecture

User App → Cloud Server → Database → Nearby Users & Police

Working Flow

  1. User registers in the application.
  2. User grants location permission.
  3. In emergency, user presses SOS.
  4. System captures GPS location.
  5. Server identifies nearby users.
  6. Alert notification is sent.

Technology Stack

  • Android Application
  • GPS Location Services
  • Firebase Backend
  • Cloud Database
  • Push Notification System

Future Scope

  • Voice-activated SOS
  • Smartwatch Integration
  • AI-based unsafe zone detection
  • Live tracking until help arrives
